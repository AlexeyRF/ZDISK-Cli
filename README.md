# ZDisk CLI

ZDisk - это инструмент командной строки для облачного хранения файлов в национальном мессенджере MAX (МАКС).

## Основные возможности

- **Шифрование:** Возможность защиты файлов паролем перед загрузкой (AES-256).
- **Поддержка больших файлов:** Автоматическое разбиение файлов на части (до 2 ГБ) для обхода ограничений и докачка при потере соединения.
- **Управление папками:** Создание виртуальной структуры папок.
- **Корзина:** Система удаления файлов с возможностью восстановления или окончательной очистки.
- **Вход по QR:** Безопасная авторизация через мобильное приложение.

## Установка

1. Склонируйте репозиторий и перейдите в папку проекта.
2. Установите зависимости и сделайте команду `zdisk` глобальной:
   ```bash
   pip install -e .
   ```
   *Теперь вы можете вызывать `zdisk` из любой папки вашего терминала.*

## Использование

1. **Настройка:**
   ```bash
   zdisk config --chat 0
   ```
   чат 0 отвечает за избранное
2. **Вход:**
   ```bash
   zdisk login
   ```
3. **Список файлов и папок:**
   ```bash
   zdisk ls               # Показать корень
   zdisk ls /documents    # Показать содержимое папки
   ```
4. **Управление папками:**
   ```bash
   zdisk mkdir /photos/vacation  # Создать папку (включая вложенные)
   ```
5. **Загрузка и шифрование:**
   ```bash
   # Загрузка в конкретную папку
   zdisk upload my_file.txt /documents
   
   # Загрузка с шифрованием AES-256 (запросит пароль или используйте -p)
   zdisk upload private.docx / -p my_password
   ```
6. **Скачивание и расшифровка:**
   ```bash
   # Если файл зашифрован, обязательно укажите пароль
   zdisk download private.docx ./downloads -p my_password
   ```
7. **Удаление:**
   ```bash
   zdisk rm my_file.txt
   zdisk rm old_file.txt --permanent
   ```
8. **Корзина:**
   ```bash
   zdisk trash --list
   zdisk trash --restore 0
   zdisk trash --clear
   ```

## Структура проекта

- `zdisk_cli.py` - Главный файл приложения.
- `zdisk_client.py` - Клиент для взаимодействия с API мессенджера.
- `zdisk_crypto.py` - Модуль для шифрования и дешифрования файлов.
- `zdisk_files.py` - Логика подготовки файлов к загрузке (разбиение/сборка).
- `file_splitter.py` / `file_assembler.py` - Инструменты для работы с частями файлов.
